Descripcion :

En este proyecto trabajos con el uso de sentencia if , ciclos for y funciones y algunas etiquetas de HTML 

El proyecto le pide datos como: Nombre, Edad 
Si cumple con la edad mínima de 21 años el programa le dará el porcentaje de Cargo  
También le preguntara si tiene cónyuge e hijos los cuales según la edad que estos tenga, al final le dará una vista detalle de todo el total que deberá pagar por el seguro 
Nota: si es menor de 18 años en programa no le dejara avanzar, ya que el requisito más importante es la edad que no sea ni menor de 18 años y ni mayor de los 65 caso contrario el programa le dejara de funcionar 

Recuerde que para responder Si es 1 y para responder no es 0 (Preguntas de hijos y cónyuge)
